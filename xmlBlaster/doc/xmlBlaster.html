This is the publish/subscribe middleware server you have needed many times before, now available for free to glue together your distributed client/server application.

Features:

  • Free for private,commercial,education use (LGPL - see LICENSE file in distribution for extensions)
  • Message Orientated Middleware (MOM)
  • Language neutral (CORBA)
  • Extensible (XML)
  • Queryable Topics (XPath or REGEX)
  • Publish/Subscribe philosophy

Requirements server side:

  • JDK 1.2 or better (Linux and other OS)
  • JacORB CORBA (Mr. Brose)
  • XSL processor (James Clark)

Requirements client side:

  • Any language supporting CORBA (C/C++, Java, Tcl, Perl, Python)
  • Any operating system

Further readings:

  • Java Message Service API (JMS) [Sun]
  • MOM informations [MessageQ]
  • EJB / JMS server [Prosyst Germany]
  • OpenMOM [Xing France]
  • iBus [SoftWired Switzerland]
  • Ozone, free Java OODBMS
